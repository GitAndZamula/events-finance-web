Коллабная - Система управления мероприятиями

## Описание проекта

Веб-приложение для управления мероприятиями, расчета финансовых показателей и генерации отчетов. Система позволяет создавать мероприятия, учитывать расходы и доходы, рассчитывать себестоимость и прибыль.

## Технологический стек

- Python 3.12+ - основной язык
- Flask 3.0+ - веб-фреймворк
- PostgreSQL - база данных
- SQLAlchemy - ORM
- Flask-Migrate - миграции
- Flask-Login - аутентификация
- Flask-WTF - формы и CSRF защита
- Bootstrap 5 - CSS фреймворк

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/gitandzamula/events-finance-web.git
cd events-finance-web
```

### 2. Создание виртуального окружения

```bash
python -m venv venv
source venv/bin/activate # Linux/Mac
venv\Scripts\activate # Windows
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Настройка переменных окружения

Создайте файл .env в корне проекта:

```env
SECRET_KEY=your-secret-key-here
DATABASE_URL=postgresql://user:password@localhost/events_finance
FLASK_APP=run.py
FLASK_ENV=development
```

### 5. Создание базы данных и миграции

```bash
# Создать базу данных в PostgreSQL
createdb events_finance

#Инициализировать миграции
flask db init
flask db migrate -m "Initial migration"
flask db upgrade
```

### 6. Инициализация БД (категории расходов и администратор)

```bash
python init_db.py
```

Этот скрипт создаст:
   - 10 стандартных категорий расходов
   - Администратора (логин: admin, пароль: admin123)

### 7. Запуск приложения

```bash
python run.py
```

Приложение будет доступно по адресу: http://localhost:5000

## Функциональность

### Для обычных пользователей

- Просмотр списка мероприятий (афиша)
- Просмотр детальной информации о мероприятии
- Переключение между отображением карточками и таблицей

### Для администраторов

- Создание, редактирование и удаление мероприятий
- Загрузка изображений для мероприятий
- Управление статьями расходов (добавление/удаление)
- Управление финансовыми расчетами:
    - Добавление расходов по категориям
    - Настройка количества участников
    - Установка стоимости участия
    - Автоматический расчет себестоимости и прибыли
- Публикация/снятие с публикации мероприятий
- Просмотр полной финансовой отчетности

### Финансовые показатели

В системе рассчитываются следующие показатели:

- Расходы - сумма затрат по всем добавленным категориям
- Себестоимость участия - расходы / количество платящих участников
- Стоимость участия - устанавливается администратором
- Доход - стоимость участия × количество платящих участников
- Прибыль/Убыток - доход - расходы

### Категории расходов (стандартные)

- Аренда помещения
- Залог в кафе
- Оплата эксперта
- Стоимость материалов
- Стоимость инструментов
- Оплата питания
- Оплата напитков
- Заработная плата
- Транспортные расходы
- Прочие расходы

Администратор может добавлять и удалять нестандартные категории

### Категории участников

- Обычные участники (платят)
- Организаторы (не платят)
- Эксперты (не платят)
- Приглашенные гости (не платят)
- Участники со скидочными картами (со скидкой)

## API Endpoints

```text
GET /api/events # Список мероприятий
GET /api/events/<id> # Детали мероприятия
GET /api/events/<id>/flyer # HTML листовка
POST /api/events/<id>/calculate # Финансовый расчет
```
